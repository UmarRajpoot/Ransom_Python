## Getting Started

### Dependencies

* Cryptography
* git
* python3

### Installing

* Clone the Repo
```
git clone '[Repo URL]'
````
* Install the Dependencies
```
pip install -r requirement.txt
````


### Executing program

* For Encryption the Sample files
```
python setup.py -e excrypt
```
* For Decryption the Sample files
```
python setup.py -d decrypt
```

## Help

Helping command
```
python setup.py -h
```
